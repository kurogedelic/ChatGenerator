では今回は、『音の三要素』について解説いたしますわ。
三要素？また“リズム・メロディ・ハーモニー”とは違うの？
それと別の“音そのものの性格を決める３つのポイント”ですの。
それが、
・音の大きさ（ボリューム）
・音の高さ（ピッチ）
・音色（おんしょく）
ですわ。
なるほど…でもそれ、ぜんぶ“なんとなく”で聴いてたかも。
では順に見てまいりましょう。まずはピッチ（Pitch）、音の高さですわ。
ピッチとは、“高い音・低い音”のことですの。
波でいえば、波が細かくなるほど高い音になりますの。
つまり、いっぱい“ピーピー”鳴るのが高い音で、
“ドーン”って響くのが低い音ってことだね！
その通り。
細かい数値はさておき、ピッチ＝高さの感覚で覚えておけば大丈夫ですわ。
次にボリューム（Volume）。
これは音の“大きさ”のことですわ。
これはわかる！
ドーンってうるさいのが大きい音、ひそひそってのが小さい音！
波の高さ（振幅）が大きいほど、大きな音に感じますの。
最後に音色（おんしょく）。これはちょっと不思議な概念ですわね。
音の“いろ”？
たとえば、同じ“ド”の音でも、
ピアノとギターじゃぜんぜん印象が違いますでしょ？
あ〜〜！それって“声”みたいなものかも！
その通り。音色とは、音の“キャラ”や“質感”のようなものですの。
物理的には、“波形の違い”が音色を作っていますのよ。
つまり、音って“高い・でかい・どんな声”って 3 つでできてるんだ！
まさにその 3 つがそろって、
“音”は個性を持つのですわ。
なるほど〜〜。
音って、思ってたよりずっと表情豊かなんだね！
ふふ。よくわかってきましたわね。
